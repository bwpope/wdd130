Easy Runs

Just get out there! The hardest part of running is just getting started! These routes will help you get moving and build your enfurance for longer runs.

Park to Park

This flat course takes you from Smith Park to Porter Park. Enjoy downtown Rexburg with plenty of places to stop and rest along the way. Both parks have running paths around them so it's easy to add on additional milage.

River Run

Get out of the traffic on this dedicated path and enjoy the scenery along the Teton River! You're bound to run into other runners on this popular trail. Great for families and ptets, this run can be extended to the Nature park for additional fun!

Temple to Temple

Run between the Rexburg Temple and the Teton River Temple. For an easy run go north to the Teton River Temple, or for a challenge run up the hill to the Rexburg Temple -or hey why not make it an out and back!
